How do I create an invoice?

  1. Click Invoices.
  2. Press New.
  1. Enter the client's name. For a new client, simply fill in the client's full name. For an existing client, a dropdown list will appear when you start typing. Select the client's name from the list.
  2. Change the date if necessary.
  3. Invoices are numbered according to the group it belongs to. Groups can be added or changed in System -> Groups.
  4. Choose the frequency for the invoice. Further options will be presented if the invoice is a recurring invoice.
  5. Press Submit.
  1. Enter a brief summary to make the invoice easier to find.
  2. If this is for a new client or if this client's information needs to be changed, click the Edit button to do so.
  3. Enter each item for your invoice. Press the Add Item button to add additional lines for your items.
  4. Enter or change the terms and conditions and/or footer area of the invoice. Defaults for these values can be set in System Settings.
  5. Review each of the items on the right side pane and adjust if necessary.
  6. Press Save.

How do I email an invoice?

  1. From the invoice edit screen, press the Email button.

** Note: The Email button will not appear on this screen unless you have configured your email settings in System Settings on the Email tab.

  1. Make sure the client's email address is filled in and accurate.
  2. Change the subject if necessary.
  3. Change the email body if necessary.
  4. Choose whether or not to attach the invoice as a PDF to the email.
  5. Press Send.

How do I enter a payment?

Payments collected manually can be entered against an invoice from the invoice edit screen.

  1. From the invoice edit screen, press the Other button.
  2. Choose Enter Payment.
  1. The amount will contain the invoice balance by default. Adjust if necessary.
  2. Change the payment date if necessary.
  3. Adjust the payment method if necessary.
  4. Optionally add a note for the payment.
  5. Choose whether or not to email a payment receipt after the payment has been entered.
  6. Press Submit.

How do I copy an invoice?

  1. From the invoice edit screen, press the Other button.
  2. Choose Copy Invoice.
  1. If the invoice is for a different client, begin entering the client's name and select the full name from the dropdown list.
  2. Change the date if necessary.
  3. Change the group if necessary.
  4. Press Submit.

How do I attach files to an invoice?

  1. From the invoice edit screen, click the Attachments tab.
  2. Click Attach Files and select the file(s) to upload.
  3. Once the files are finished uploading, optionally set whether or not you want them to be visible to the client when they view the public invoice URL.

How do I create custom fields for invoices?

There are certain fields which are useful for some but not all. For example, you may have a need to include a purchase order number on your invoices. This is a good example of how custom fields can be helpful.

  1. Press the System icon in the top-right menu bar.
  2. Select Custom Fields.

Why can't I change the status to paid?

The paid status is the only status you cannot manually change an invoice to. To change an invoice to Paid status, the invoice must have a payment made in full. Once the invoice has no remaining balance, the status will automatically update to Paid.

How do I set up an existing invoice as a recurring invoice?

  1. From the invoice edit screen, press the Other button.
  2. Choose Recur Invoice.
  1. Enter the date that the first recurring invoice should be generated.
  2. Choose the frequency.
  3. Optionally enter an end date for the recurring invoices.
  4. Press Submit.

** Note: A cron job is required to check for and create recurring invoices. Be sure your cron job is set up properly.

Why aren't my recurring invoices working?

First, check the Next Date of the recurring invoice you expect should have generated.

  1. Click Recurring Invoices.
  2. The date in the Next Date field indicates the date which this invoice should recur next. If this date is in the future, then the invoice isn't ready to recur yet.

If the Next Date is today's date or prior to today's date but the recurring invoice hasn't been generated, then the next step would be to visit http://YourFusionInvoiceURL/tasks/run (or http://YourFusionInvoiceURL/index.php/tasks/run if you have to specify index.php in your URL).

One of two things will happen when you visit this URL in your browser:

  1. You'll be greeted by an empty, white page. This is good - if you go back to check your list of recurring invoices, you should find that the Next Date has incremented to the next date in the set frequency. You should also find that the new invoice has appeared in your list of invoices. If this is the case. then the Task Scheduler cron job hasn't been set up or has been set up improperly.
  2. You'll be greeted with a lovely, "Whoops..." error message. If this is the case, there will be details logged at the bottom of your storage/logs/laravel.log file.
