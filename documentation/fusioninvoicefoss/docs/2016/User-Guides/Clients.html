How do I add a client?

  1. Click Clients.
  2. Press New.
  1. Enter the client's name.
  2. Enter the client's email address.
  3. Choose whether or not to allow the client to log in. When allowing a client to log in, the client will use their email address and the password you enter to log in.
  4. Enter the client's address. The entire address can be entered in the Address field, or can be split out into the separate city, state, psotal code and country fields.
  5. Choose the client's currency.
  6. Choose the client's language.
  7. Press Save.

How do I create custom fields for clients?

There are certain fields which are useful for some but not all. For example, a lot of people need to include their client's VAT ID on their invoices. This is a good example of how custom fields can be helpful.

  1. Press the System icon in the top-right menu bar.
  2. Select Custom Fields.
  1. Press New.
  1. Select Clients from the Table Name dropdown list.
  2. Enter the label for the field being created.
  3. The Field Type can be Text, Dropdown or Textarea. Select Text for a simple text field. Select Dropdown for a dropdown list. Select Textarea for a larger text / memo field.
  4. If Dropdown was selected in the Field Type field, enter the values which should appear in the dropdown separated by commas. For example, if the dropdown should contain a list of the colors red, blue and orange, then you would enter: Red,Blue,Orange
  5. Press Save. Once saved, the new custom field will appear on the client edit screen.

How do I create logins for my clients?

You may optionally create login accounts for your clients to use to view all of their invoices, quotes, payments, etc. A client login account is not required for the client to pay their invoices online. It is only required to give clients access to all of their documents in a convenient fashion.

  1. From the client edit screen, select Yes under Allow Login.
  2. Enter the password the client should use when logging in.
  3. Re-enter the password again for confirmation.
  4. Press Save. The client will log in at the same URL as you do but using their email address and the password entered above.

How do I change the default currency or language for a client?

Your clients may use different currency or speak another language from your own. In this case, you can adjust your client's record to reflect their local currency and native language.

  1. From the client edit screen, select the client's currency.
  2. Select the client's language.
  3. Press Save.
