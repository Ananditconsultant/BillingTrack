Upgrade

** Note: This documentation is applicable for updating an existing FusionInvoice 2016 installation. If you are upgrading from FusionInvoice v2, please use the Upgrading from FusionInvoice v2 documentation instead.

Step 1: Download the latest package

Step 2: Unzip the package

Step 3: Upload the files to your server

Step 4: Complete the upgrade

Step 1: Download the latest package

  1. Download the latest package. Save it locally to your computer.

Step 2: Unzip the package

Navigate to the downloaded install package and unzip the contents.

Step 3: Upload the files to your server

Upload the following folders to the FusionInvoice folder on your server and let them merge/overwrite the existing folders:

  • app
  • assets
  • database
  • resources
  • vendor (only required to upload if you are upgrading to version 2016-15 or higher from a version below 2016-15)

It is recommended that you use a standard FTP program such as FileZilla to upload the files to your server.

Step 4: Complete the upgrade

Visit your FusionInvoiceURL/setup in your web browser. If your FusionInvoiceURL/setup produces an error, try using FusionInvoiceURL/index.php/setup instead. This step will upgrade your database. Once this step is completed, you should be able to log back into your upgraded FusionInvoice system.
