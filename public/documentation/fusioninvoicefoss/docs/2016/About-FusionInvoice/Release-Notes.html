Release Notes

FusionInvoice 2016-20 (Dec 14, 2016)

  • Enhancements and General Changes
    • Added UserCreated, UserDeleted and UserSaving event listeners.
  • Bugfixes
    • Fixes a bug where default custom records were not being created as needed for payments.
    • Fixes a bug where default custom records were not being created as needed for clients.
    • Fixes a bug where default custom records were not being created as needed for quotes and invoices.
    • Fixes a bug when creating quotes.
    • Fixes a bug when using the domPDF driver with PHP 7.1.
    • Payments can no longer be made on canceled invoices.
    • Fixes translation encoding issues on some confirmation windows.

FusionInvoice 2016-19 (Dec 6, 2016)

  • Bugfixes
    • Fixes a bug with the Stripe payment modal form.

FusionInvoice 2016-18 (Dec 5, 2016)

  • Enhancements and General Changes
    • Custom fields can now be included when importing client records.
    • Invoice and quote summary dashboard widgets now allow time span configuration directly from the widget.
    • Adds basic validation to System Settings.
    • Adds discount column to Client Statement report.
    • The listPayments API method now accepts optional invoice_id, invoice_number and client_id key=>value parameters. Note: A new fiapi.php file is required for those using the API.
    • Event for ClientCreated now fires after a client has been created.
  • Bugfixes
    • Fixes a bug which prevented quotes and invoices with a forward slash in the invoice/quote number from being emailed.
    • Changed the "reports" route prefix to "report" to prevent conflicts in IIS where "reports" might be reserved for SSRS.
    • Fixed a bug in the Profit & Loss report.
  • Translations
    • Updated Spanish translation (thank you, Alyssa).

FusionInvoice 2016-17 (Sept 19, 2016)

  • Enhancements and General Changes
    • Tax rates can be defined on Item Lookups.
    • Expenses can be filtered by category, vendor and status.
    • Expenses can now be billed to invoices without having to add a line item.
    • Expenses can be exported.
    • Adds settings for quantity / amount decimals and decimals for tax rounding.
  • Bugfixes
    • Fixes a bug searching expenses when a database table name prefix is in use.
    • Client balances no longer include canceled invoice amounts.
    • Fixes a bug preventing the user from being recorded when creating expenses.
    • Fixes a bug which caused the import utility to ignore the quote / invoice status id.
  • Translations
    • Updates French translations.
    • Deletes translations which were not being maintained.

FusionInvoice 2016-16 (Sept 5, 2016)

  • Enhancements and General Changes
    • Expenses can be searched.
    • Custom textarea fields are now autosized.
    • Invoice templates can use $invoice->amount->formatted_total_without_conversion and quote templates can use $quote->amount->formatted_total_without_conversion to retrieve the total prior to currency conversion.
  • Bugfixes
    • Corrects column type for expense amount.
    • Fixes a bug causing the Client Statement report to error when no results are available.
    • Fixes a redirect issue.
  • Translations
    • Updates Turkish translations.

FusionInvoice 2016-15 (Aug 18, 2016)

*** Important note: The FusionInvoice 2016-15 update requires an additional folder (the vendor folder) to be uploaded to your server along with the normal four folders during step 3 of the upgrade documentation.

  • Enhancements and General Changes
    • Quote and invoice importers will now import the summary field if it exists in the import file.
    • Taxes now calculate using 3 decimals.
    • Custom text can be specified for payment buttons on the public invoice page.
    • Add-on language files must be referenced using trans('AddonFolderName::filename.language_key')
    • Updates framework and vendor dependencies.
  • Bugfixes
    • Fixes a bug when editing payments.
    • Fixes a bug which reported the invoice total amount as the payment amount in the list of payments in the Client Center.
    • Fixes a bug which caused an error when editing expenses with attachments.
  • Translations
    • Updates German translations.

FusionInvoice 2016-14 (Jun 19, 2016)

  • Enhancements and General Changes
    • Email subjects are now fully customizable.
    • Adds the ability to automatically email upcoming invoice payment notices.
    • Groups are now fully customizable and has support for optional number resets yearly, monthly and weekly.
    • Adds links to invoice and client from payments screen.
  • Bugfixes
    • Fixes a bug where some clients / quotes / invoices wouldn't obey the default template setting in System Settings.
    • Fixes a tax currency display issue on quote and invoice edit screens.
    • Fixes hard coded language reference.
    • Fixes offsite CSS reference on login page.
    • Fixes a bug preventing terms from being included when importing quotes and invoices.
  • Translations
    • Updates Dutch translations.
    • Updates German translations.
    • Updates Slovak translations.

FusionInvoice 2016-13 (Apr 27, 2016)

  • Bugfixes
    • Fixes a bug causing errors when editing an existing expense and when attempting to bill an expense.

FusionInvoice 2016-12 (Apr 26, 2016)

  • Enhancements and General Changes
    • Adds new Expense module for expense tracking.
    • Adds new Profit and Loss report.
    • Files can now be attached to invoices, quotes, expenses and clients.
    • Percentage based discounts can now be added to quotes and invoices.
    • Adds options to System Settings -> Online Payments tab to control which fields should be displayed and/or required when using Stripe to collect online payments.
    • Placed some content on invoice and quote edit screens within tabs.
    • Adds Payments tab to invoice edit screen.
  • Bugfixes
    • The send email validator will no longer fail to validate multiple email addresses entered in the To / Cc / Bcc boxes when they're separated by commas but also spaces.
  • Translations
    • Updates French translations.
    • Updates Icelandic translations.
    • Updates Norwegian translations.

FusionInvoice 2016-11 (Mar 26, 2016)

  • Enhancements and General Changes
    • When a large logo image is uploaded, it will now display with a maximum width of 200px in System Settings (although the actual dimensions of the image isn't changed).
    • LastPass should no longer attempt to automatically populate the email address on the client edit screen with your own email address.
    • Stripe payment form now includes and requires billing name, billing address and billing zip code.
  • Bugfixes
    • Fixes a bug which caused the system menu to appear "blank" on some mobile devices / resolutions.
    • Fixes a bug which prevented PDF reports from opening in Safari.
    • Fixes a bug which would convert amounts based on applying the exchange rate incorrectly when the from / to currencies used different formats.

FusionInvoice 2016-10 (Mar 14, 2016)

  • Enhancements and General Changes
    • Invoices which are converted from quotes will now display the relationship to the quote on the invoice edit screen.
    • Quotes which are converted to invoices will now display the relationship to the invoice on the quote edit screen.
    • Adds image($imagePath, $width, $height) helper function so images (other than logos) may easily be added to custom templates.
    • Adds the ability to override the default HTML email template layout by creating a file at the following location: /custom/templates/email_templates/html.blade.php
    • Adds Create Invoice and Create Quote links to the default dashboard widgets for quick access.
  • Bugfixes
    • Fixes a bug in the Client Statement Report which prevented the report from running when a client had an ampersand in their name.
    • Fixes a bug which would cause a dashboard widget to "disappear" if multiple widgets had the same order number on the Dashboard tab of the System Settings screen.
  • Translations
    • Adds Slovak translation.
    • Adds Norwegian translation.
    • Updates German translation.
    • Updates French translation.

FusionInvoice 2016-9 (Feb 21, 2016)

  • Enhancements and General Changes
    • A prompt now appears when attempting to delete quote / invoice items to prevent accidental record deletion.
  • Bugfixes
    • Fixes a bug which recorded online payment dates incorrectly.
    • Fixes a problem where setting an existing invoice to recur would not record the stop date even if the date was provided.
    • Fixes a bug which produced an error message when a client would approve a quote through the quote's public URL.
    • Fixes a bug where the payment method was not outputting on the Payments Collected report.
    • Fixes a bug which caused items added to a quote to always create an item lookup record, even when the Save Item As Lookup check box wasn't checked.
  • Translations
    • Updates French translation.

FusionInvoice 2016-6 (Jan 30, 2016)

  • Enhancements and General Changes
    • Adds Logo::size(width, height) function to allow for higher quality logos to display on PDF files.
    • Outgoing email no longer requires cron job before it will be sent. *** You MUST be sure to adjust your cron jobs to run only once per day instead of once per minute.
    • The Notes button on the public quote / invoice screen now toggles based on whether the quote/invoice or the notes are being viewed.
    • Remove requirement for install and uninstall methods for add ons.
    • The AddonServiceProvider no longer attempts to load resources from add ons if they're not installed.
  • Bugfixes
    • Fixes bugs in Import module which would prevent quotes and invoices from importing.
    • Fixes several namespacing problems in the API module.
    • Fixes a bug where editing a client record from the invoice or quote edit screen would display empty custom field values.
    • Fixes a problem where new client records would not inherit proper default template settings.
    • Fixes an issue where the public invoice/quote page layout was not translating to the client's specific language.
  • Translations
    • Adds Icelandic translation.
    • Updates French translation.
