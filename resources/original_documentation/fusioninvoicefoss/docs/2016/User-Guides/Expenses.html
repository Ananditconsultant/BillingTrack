How do I enter an expense?

  1. Click Expenses.
  2. Press New.
  1. Choose the date of the expense.
  2. Enter the category to assign the expense to. If you have previously entered the same category, it will appear once you start typing.
  3. Enter the amount of the expense.
  4. Optionally, enter the name of the vendor for the expense. If you have previously entered the same vendor, it will appear once you start typing.
  5. If you wish to bill the expense to a client, enter the name of the client. This will allow you to add this expense to an invoice after the expense is saved.
  6. Optionally, enter a description for the expense.
  7. Optionally, select one or more files to attach to the expense.
  8. Press Save.

How do I bill an expense to a client?

  1. Click Expenses.
  2. Press Options.
  3. Select Bill This Expense.
  1. Choose the invoice to add the expense to.
  2. Enter the name of the line item to add to the invoice. This value defaults to the category of the expense but can be changed to whatever you would like.
  3. Enter a description for the line item to add to the invoice. This value defaults to the description of the expense but can be changed to whatever you would like.
  4. Press Submit.

How can I see my profit and loss?

  1. Click Reports.
  2. Choose Profit and Loss.
  1. Choose the from date for the report.
  2. Choose the to date for the report.
  3. Choose whether you want to preview the report or open it as PDF.
  4. Press Run Report.
